Micro Facebook Clone

Powered by Django + React + Ant Design UI Kit + Heroku
It uses multiple features and libraries of Django and React

Django REST Framework

It provides the frontend the data from the database through API endpoint with the help of Django REST Framework

Django Token Authentication

It uses Token Authentication for the log-in process of the frontend and checks the User database at the backend.

Django Custom User Class

It extends UserBaseManager class to create Custom User to adjust specific needs of this apps. It also customize the UserAdmin class for the same reason.

React Router

It handles the routing process at the frontend using React Router.

Axios

It makes HTTP Request from the client browser to the backend using Axios Library.

React Components

Its frontend display consist of dozens of React Components with Ant Design UI Kit.

PostgreSQL Database

It changes its default database (SQLite3) to PostgreSQL database to adjust the production environment (Heroku app).

Mini CMS

Powered by Django + Bootstrap + Heroku

CRUD Posts

It enables user to create, display, update, and delete posts

Login Logout

It provides user registration page and allows to login and logout.

User Profile

It allows user to change their profile picture. It uses Amazon AWS S3 and django-storages library to serve static file.

Email Password Reset

It enable users to request password reset when forget their password. They will receive notification through email. It uses gmail SMTP to send email.

Pagination

It gives pagination to blog posts.

Material Design Bootstrap
